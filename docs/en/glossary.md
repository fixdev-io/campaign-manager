# 📚 Glossary

- **Campaign** — a flow of messages sent to a user group.
- **Segment** — a set of users matching filters.
- **Delivery** — message sending process.
- **Policy** — contact restrictions.
- **Workflow** — orchestrated steps in Temporal.
